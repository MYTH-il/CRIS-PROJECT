# Real Data Drop Zone (Not Committed)

Place any real datasets in this folder for local experimentation. Suggested structure:

- `ncrb/` — NCRB tables (CSV) by year or category
- `rti/` — RTI-obtained FIR summaries (CSV/JSONL)
- `judgments/` — Indian Kanoon / JUDIS text dumps (.txt)

Do not commit real data to version control.
